infinite asynchronous rock/paper/scissors help

r => rock
p => paper
s => scissors

can submit multiple turns in one command, which is useful if you're a new player. eg:
'rrrrpsrpsrpssspssr'
sequences will be trimmed to the current maximum sequence length of all players, plus one (to gradually advance the number of rounds)

'reset' resets your own sequence (start again as a new player)
